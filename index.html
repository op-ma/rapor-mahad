Selamat Datang di Aplikasi Rapor v4.0

Aplikasi ini dirancang untuk memudahkan manajemen nilai dan pencetakan rapor siswa secara luring (offline) langsung di peramban (browser) Anda.

💡 Silakan Login (tanpa password) terlebih dahulu melalui tombol di pojok kanan atas untuk membuka fitur-fitur aplikasi.

Langkah Pengisian:

  1. Identitas Madrasah: Lengkapi profil madrasah, nama Kepala Madrasah, Wali Kelas, dan info semester.
  2. Mata Pelajaran: Atur mapel akademik dan praktik. Anda dapat mengimpor dari file Excel.
  3. Data Siswa: Masukkan data siswa secara manual atau melalui *import* Excel. Pastikan NISN unik.
  4. Input & Rekap Nilai: Masukkan nilai di sini. Anda bisa mengetik langsung di tabel atau mengunggah nilai via Excel. Gunakan fitur pencarian untuk menemukan siswa dengan cepat.
  5. Cetak Rapor: Pratinjau nilai per siswa atau seluruh siswa secara bersamaan, lalu klik tombol Cetak.

Keamanan & Penyimpanan Data

Seluruh data yang Anda masukkan tersimpan secara lokal di *browser* perangkat ini. Jika Anda menghapus riwayat penjelajahan (*clear cache/data*), data Anda bisa hilang.

Rekomendasi Penting:

Gunakan fitur Backup Data yang berada pada bar kuning di atas secara rutin. File JSON backup dapat disimpan dengan aman, dan kapan pun bisa dikembalikan dengan fitur Restore Data.
